import { Aside, LinkCard, Steps } from '@astrojs/starlight/components';
import CommandInput, { OptionType } from '../../../components/CommandInput.astro';

This guide will teach you how to only allow Moderators to star messages.

---

<Steps>

0.  If you haven't already, set up the bot.

    <LinkCard title="Getting Started" href="../getting-started"/>

1.  Blacklist everyone:

    <CommandInput command="blacklist add role" options={[
        {
            name: 'role',
            value: 'everyone',
            type: OptionType.Role
        }
    ]}/>

2.  Whitelist your Moderator role:

    This will allow your Moderators to bypass the blacklist.

    <CommandInput command="whitelist add role" options={[
        {
            name: 'role',
            value: 'Moderators',
            type: OptionType.Role
        }
    ]}/>

3.  Filter blacklisted members:

    This will make it so that reactions by blacklisted members won't be counted by Starboard.

    <CommandInput command="settings edit filtering" options={[
        {
            name: 'filter-blacklisted-users',
            value: 'Filter Reactors',
            type: OptionType.String
        }
    ]}/>

    <Aside type="note">
        This will allow Moderators to star messages by blacklisted members.
        
        If you only want to allow Moderators to star messages by Moderators, you can set it to filter both Reactors and Message Authors.
        <CommandInput command="settings edit filtering" options={[
        {
            name: 'filter-blacklisted-users',
            value: 'Filter Both (Reactors & Message Authors)',
            type: OptionType.String
        }
    ]}/>
    </Aside>

4.  (Optional) Remove invalid reactions:

    <Aside type="tip">
         If you wish, you can optionally make Starboard remove invalid reactions, such as reactions from blacklisted members.
         <CommandInput command="settings edit filtering" options={[
            {
                name: 'remove-invalid-reactions',
                value: 'True',
                type: OptionType.Boolean,
            }
         ]}/>
    </Aside>

</Steps>

That's it!  Now only Moderators can star messages.
