## Using a Custom Bot
Have a special community bot or a server theme to follow? You can configure a custom bot to use to post starboard messages, while the magic of Starboard works in the background.

<Aside type="note">
   Using a custom bot requires your server to have <ExternalLink type={LinkType.Premium}/>.
</Aside>

<Aside type="note" title="Things to note">
- The custom bot is only used for posting and editing starboard messages.
  - Everything else (listing for reactions, commands, auto-star, reward roles, etc) is still handled by Starboard itself.
  - Consequently, this means you still need to have Starboard in your server.

- All custom bot tokens are stored encrypted, and will not be visible to anyone (including you).
</Aside>

<Steps>

1. Create a new application on <ExternalLink type={LinkType.Other} href="https://discord.com/developers/applications" title="Discord's Developer Portal"/>.
    <Image class="rounded-xl shadow-lg" src={createApplicationImage} alt="Discord's Application Creation flow." />

2. Invite the bot to your server.

    <Aside type="tip" title="Hint">
        You can copy your bot's application ID and paste it below to generate the recommended invite link.
        <Image class="rounded-xl shadow-lg" src={copyApplicationIDImage} alt="How to copy the bot's application ID." />
       <InviteGenerator/>
        
    </Aside>

3. Go the bot tab of your newly created application, and copy its token (you may need to reset it first).
     <Image class="rounded-xl shadow-lg" src={copyTokenImage} alt="How to copy the bot's token." />

4. Set your custom bot's token.
    <CommandInput command="custom-bot set" options={[
    {
        type: OptionType.String,
        name: 'token',
        value: "MTM0ODQyNz...",
    }
    ]}
   />

</Steps>

And that's it! Starboard messages will now be posted by your custom bot.
